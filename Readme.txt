Um site com os recursos de HTML, PHP e um pouco de CSS. página inicial é lista todas as marmitas que Dona Rita produz utilizando uma estrutura de repetição. É necessário que os dados: nome da marmita, tamanho, preço, imagem e ingredientes, estejam em um array multidimensional.
Ao clicar em uma marmita, a mesma será aberta em uma outra página contendo todas as informações da marmita selecionada. Para isso você deverá passar o código da marmita e busca-lo no array utilizando a estrutura de repetição e condição para exibir apenas os dados da marmita selecionada, além apresentar os dados para contato com a dona Rita.
